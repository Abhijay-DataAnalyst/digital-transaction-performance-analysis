## Overview
This project focuses on analyzing transaction-level data in the financial services domain
to understand spending behavior, transaction trends, and performance across banks.
The analysis combines Python-based web scraping, data cleaning, and Power BI dashboards.

## Data Collection
- Collected transaction-related datasets using **Python (BeautifulSoup)**
- Automated downloading of RBI-related reports and datasets
- Cleaned and structured data using **Excel** for analysis readiness

## Tools & Technologies
- Python (BeautifulSoup, pandas)
- Microsoft Excel
- Power BI

## Analysis & Dashboards
- Built interactive **Power BI dashboards** to track:
  - Transaction volumes
  - Cards issued
  - Spending trends
  - Average transaction size
- Compared performance across multiple banks
- Identified seasonal trends and behavioral patterns

## Key Insights
- Transaction volumes and spending show strong seasonal patterns
- Certain banks demonstrate higher spending per card despite lower issuance
- Data-driven insights support performance optimization and targeted strategies

## Deliverables
- Power BI dashboard for monitoring transaction KPIs
- Python script for automated data collection
- Cleaned datasets (sample only)

## Notes
Raw transaction data and full reports are not included due to data sensitivity.
This repository focuses on methodology, analysis approach, and business insights.
